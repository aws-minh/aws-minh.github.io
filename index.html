Power BI Sales Analytics Dashboard

View Dashboard

A comprehensive Power BI sales analytics solution that tracks $5.63M in revenue across 94K orders, monitoring performance across 2,681 brands and 27.9K products in 14 nations. The dashboard provides real-time insights into sales trends, customer demographics, regional performance, and product category analysis with interactive visualizations.

Key Features & Analytics:
  • Multi-dimensional sales tracking with time-series analysis showing growth trends from 2020-2022, including year-over-year comparisons and seasonal pattern identification across global markets
  • Advanced customer segmentation by age groups (12-19, 20-39, 40-59, 60+) with detailed demographic analysis revealing spending patterns and revenue contribution by segment, enabling targeted marketing strategies
  • Geographic performance visualization across 14 countries with category-specific sales breakdowns for Outerwear & Coats, Jeans, Sweaters, Suits & Sport Coats, and Swim categories, providing regional market insights
  • Order status monitoring with real-time tracking of cancelled, complete, processing, returned, and shipped orders for operational insights and supply chain optimization
  • Interactive drill-down capabilities enabling users to explore sales data by margin growth, product performance, and regional trends with dynamic filtering and cross-highlighting features

Live Iran War Tracker

A comprehensive real-time conflict monitoring platform that tracks and visualizes Iran conflict events, news, and statistics through an interactive dashboard. This full-stack application combines React frontend with Leaflet maps and real-time data feeds, powered by a Flask backend with automated data collection from multiple sources.

Key Features & Architecture:
  • Engineered a real-time conflict monitoring platform using React frontend with interactive Leaflet maps, dynamic charts using Recharts, and live data feeds for tracking conflict events, news updates, and statistical analysis
  • Implemented a sophisticated 3-container Kubernetes architecture with nginx reverse proxy for intelligent traffic routing, separating frontend and API traffic, integrated with Cloudflare tunnel for secure external access and SSL termination
  • Developed production-grade DevOps infrastructure featuring containerized microservices, infrastructure-as-code with Kubernetes manifests, automated CI/CD pipelines, persistent volume management for SQLite data storage, and comprehensive monitoring with health checks
  • Built automated data fetching systems with Flask backend for real-time information aggregation from multiple conflict monitoring sources, implementing cross-origin resource handling and robust error management
